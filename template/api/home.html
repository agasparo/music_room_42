Lyrics

			

			

				L'api music room a une partie consacree au lyrics, il y a deux routes possibles :
				

				
1 * Recuprer tout les lyrics

				

				
Les parametre(s)

				

				  	
				    	

				    	  	Nom
				    	  	Type
				    	  	Description
				    	

				  	
				  	
				    	

				      		myToken
				      		string
				      		votre token
				    	

				    	

				    		apiKey
				    		string
				    		votre api key
				    	

				   	
			   	

			   	

			   	
Le retour

				

				  	
				    	

				    	  	Nom
				    	  	Type
				    	  	Description
				    	

				  	
				  	
				    	

				      		content
				      		Json
				      		
				      			Le retour est un json avec en parametres :

				      			- status : int (200 si tout est ok)

				      			- number_results : int le nombres de resultats trouves

				      			- response_time : float le temps de reponse de l'api

				      			- encode : array si il y a une encodage, il contiendra la fonction de d'encodage

				      			- response : array contient la reponse
				      				
				    	

				   	
			   	

			   	

			   	
Essayer

			   		Pour tester le code clicker ici et utiliser la fenetre de test
			   	

			   	
Le code

				
const url = "http://localhost/music_room/api/lyrics/all";
$.get(url, {
	myToken: "{ yourToken }",
	apiKey: "{ yourApiKey }"
}, function (content) {
	
	console.log(content);

});
